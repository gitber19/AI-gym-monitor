# 🚀 Quick Start Guide - AI Gym Trainer Pro

## How to Open the Web Application

### ⚡ Method 1: Simple (Double-Click)
1. Navigate to the `project ptiot` folder
2. Double-click `index.html`
3. The app will open in your default browser

### 🌐 Method 2: Using Python Server (RECOMMENDED)
For best results and camera access:

1. **Open Command Prompt or PowerShell**
   - Press `Win + R`
   - Type `cmd` or `powershell`
   - Press Enter

2. **Navigate to the project folder**
   ```bash
   cd "C:\Users\Dell\Desktop\sem 3\coding shit\project ptiot"
   ```

3. **Run the server**
   ```bash
   python start_server.py
   ```
   
   Or simply double-click `start_server.bat`

4. **Open in browser**
   - The browser should open automatically
   - Or manually go to: `http://localhost:8000/index.html`

### 📋 Step-by-Step Usage

1. **First Time Setup**
   - Allow camera access when prompted
   - Go to **Profile** tab
   - Fill in your details (name, age, weight, height, etc.)
   - Click **Save Profile**

2. **Start Workout**
   - Go to **Workout** tab
   - Click on any exercise (Squats, Push-ups, etc.)
   - Position yourself in front of camera
   - The app will automatically detect your pose and count reps!

3. **View Your Routine**
   - Go to **Routine** tab
   - See personalized workout based on your BMI

4. **Check History**
   - Go to **History** tab
   - View all your past workouts

## 🔧 Troubleshooting

### Camera Not Working?
- Make sure you're using the server method (Method 2)
- Check browser permissions (allow camera access)
- Try a different browser (Chrome recommended)

### Browser Compatibility
- ✅ Chrome (Best)
- ✅ Firefox
- ✅ Edge
- ❌ Internet Explorer (Not supported)

### Pose Not Detected?
- Ensure good lighting
- Stand 6-8 feet from camera
- Make sure full body is visible
- Use a plain background

## 📝 Notes
- All data is saved locally in your browser
- No internet connection required after initial page load
- Camera access is required for pose detection

## 🎯 Quick Tips
- Stand in front of a plain wall for best results
- Ensure good lighting
- Wear contrasting clothes for better detection
- Complete full range of motion for accurate rep counting
